Under the Hood

Technical Architecture

📊

Data Pipeline

Live stock data fetched via yfinance API. Computes annualised returns and covariance matrix from 252 trading days of historical closing prices.

📐

Classical Solver

Markowitz Mean-Variance optimization via SciPy's SLSQP. Minimises portfolio variance while maximising expected return, with diversification bounds [5%–40%].

⚛️

Quantum Solver

Portfolio reformulated as a QUBO problem. Solved using IBM Qiskit's QAOA ansatz (p=1) with COBYLA optimizer on a StatevectorSampler simulator.

🧮

QUBO Formulation

Binary variables qi ∈ {0,1} per asset. Objective: minimise λ·qTΣq − (1−λ)·μTq where Σ is the covariance matrix and μ is expected returns.
